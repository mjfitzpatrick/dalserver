SSAP Parameters
VERSION[STRING]SSAP protocol version
REQUEST[STRING]Operation to be performed
POS[FLOAT, RANGELIST]Central coordinates of search region
SIZE[FLOAT]Size (diameter) of the search region
BAND[ORDERED, RANGELIST]Spectral bandpass of the search region
TIME[ISODATE, ORDERED, RANGELIST]Range of times for the search region
FORMAT[STRING, RANGELIST]Allowable output data formats
RESPONSEFORMAT[STRING]Format of query response
APERTURE[FLOAT]Aperture diameter for spectral extraction
SPECRES[FLOAT]Minimum spectral resolution
SPECRP[FLOAT]Spectral resolving power
SPATRES[FLOAT]Minimum spatial resolution
TIMERES[FLOAT]Minimum temporal resolution
SNR[FLOAT, ORDERED, RANGELIST]Minimum signal to noise ratio
Redshift[FLOAT, ORDERED, RANGELIST]Redshift range
VarAmpl[FLOAT, ORDERED, RANGELIST]Variability amplitude range
TargetName[STRING]Target name
TargetClass[STRING, RANGELIST]Target class names to search for
FluxCalib[STRING]Spectrum is flux calibrated
WaveCalib[STRING]Spectrum is wavelength calibrated
PubDID[STRING]Publisher-assigned dataset identifier
CreatorDID[STRING]Creator-assigned dataset identifier
Collection[STRING, RANGELIST]Data collection name or pattern
Top[INTEGER]Number of top-ranked items to return
Maxrec[INTEGER]Maximum number of output records
Mtime[ISODATE, ORDERED, RANGELIST]Range of modification times
Compress[BOOLEAN]Allow dataset compression
Verb[INTEGER]Output verbosity level
RunID[STRING]Runtime job ID string
